font funcion

2. Font

los fonts son herramientas fundamentales en el diseño gráfico y la comunicación visual. No solo embellecen el contenido, sino que también mejoran la legibilidad, comunican emociones y establecen una identidad de marca. 

	<"!DOCTYPE html>
<"html lang="es">
<"head>
    <"meta charset="UTF-8">
    <"meta name="viewport" content="width=device-width, initial-scale=1.0">
    <"title>Ejemplo de Fuente<"/title>
    
    <"link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
    <"style>
        body {
            font-family: 'Roboto', sans-serif;
        }
    <"/style>
<"/head>
<"body>
    <"h1>Hola, Mundo!<"/h1>
    <"p>Este es un ejemplo de texto usando la fuente Roboto.<"/p>
<"/body>
<"/html>

Los fonts permiten controlar la apariencia visual del texto en una página web, incluyendo el tipo de letra, el tamaño, el estilo (normal, cursiva, negrita), el color, etc.
