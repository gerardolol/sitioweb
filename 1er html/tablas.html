13.tablas

Las tablas en HTML se crean utilizando la etiqueta <"table>, que permite organizar datos en filas y columnas. Las tablas son útiles para presentar información estructurada de manera clara y accesible. A continuación, se describen las etiquetas principales que se utilizan para crear tablas:

		
		
			<"table>
  <"tr>
    <"th>Encabezado 1<"/th>
    <"th>Encabezado 2<"/th>
    <"th>Encabezado 3<"/th>
  <"/tr>
  <"tr>
    <"td>Dato 1<"/td>
    <"td>Dato 2<"/td>
    <"td>Dato 3<"/td>
  <"/tr>
  <"tr>
    <"td>Dato 4<"/td>
    <"td>Dato 5<"/td>
    <"td>Dato 6<"/td>
  <"/tr>
<"/table>
		
		
<2table>: Define el inicio de la tabla.

		
<"tr>: Define una fila de la tabla.

		
<2th>: Define una celda de encabezado, que generalmente está en negrita y centrada.

		
<"td>: Define una celda de datos.
