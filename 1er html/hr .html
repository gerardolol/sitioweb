Los hr sirven para colocar un espacio entre los texto dentro de ellos y que el texto se muestre en la pagina 
		
		
			
			<"style>
    hr {
      border: none; /* Eliminar el borde predeterminado */
      height: 1px; /* Establecer el grosor de la línea */
      background-color: #ccc; /* Color de la línea */
      margin: 30px 0; /* Espacio alrededor de la línea */
    }
  <"/style>
